########################
Stratis Sidechains
########################

As described in the Architecture Reference section, a Stratis Sidechain allows for simple deployment of customised and tuned blockchains. These chains hold a relationship with the Stratis mainchain, with the issued token being pegged to the STRAX.

Guides contained within this section will relate to operations that can be achieved with Stratis Sidechains.
